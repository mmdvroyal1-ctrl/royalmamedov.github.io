🎨 CSS Tam Bələdçi

CSS nədir?

CSS (Cascading Style Sheets) veb saytın görünüşünü idarə edir. HTML strukturu yaradırsa, CSS onu gözəlləşdirir.

👨‍💻 Kim yaradıb?

CSS 1996-cı ildə 0 tərəfindən yaradılıb.

🎯 Niyə yaradılıb?

HTML əvvəl dizayn üçün istifadə olunurdu, amma bu çox çətin idi. CSS yaradıldı ki, dizayn ayrıca idarə olunsun.

🎨 Əsas xüsusiyyətlər

  • Rənglər
  • Fontlar
  • Layout
  • Animasiyalar

💻 Nümunə kod

body {
  background:black;
  color:white;
}

h1 {
  color:red;
}

📐 Flexbox (layout sistemi)

.container {
  display:flex;
  justify-content:center;
  align-items:center;
}

✨ Animasiya

.box {
  transition:0.5s;
}

.box:hover {
  transform:scale(1.2);
}

🚀 CSS ilə nə edə bilərsən?

  • Modern UI
  • Mobil dizayn
  • Animasiya effektlər
  • Responsive sayt
