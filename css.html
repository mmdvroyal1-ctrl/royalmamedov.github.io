📜 CSS nədir?

CSS (Cascading Style Sheets) veb saytların dizaynını idarə edən dildir.

👨‍💻 Kim yaradıb?

CSS 1996-cı ildə Håkon Wium Lie tərəfindən yaradılmışdır.

🎯 Məqsəd

Saytın görünüşünü gözəlləşdirmək və dizayn vermək.

🧱 Sintaksis

selector {
  property: value;
}

🎨 Rənglər

color: red;
background: black;

📏 Ölçülər

width:100px;
height:200px;

📦 Box Model

margin:10px;
padding:10px;
border:1px solid white;

📐 Flexbox

display:flex;
justify-content:center;
align-items:center;

📱 Responsive

@media(max-width:600px){
  body{background:red;}
}

✨ Animasiya

@keyframes glow{
 from{opacity:0;}
 to{opacity:1;}
}

🚀 Real nümunə

button{
  background:#22c55e;
  padding:10px;
  border-radius:10px;
}
