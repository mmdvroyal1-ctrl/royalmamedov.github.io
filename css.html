🎨 CSS Tam Bələdçi

CSS nədir?

CSS (Cascading Style Sheets) saytın dizaynını yaradır. HTML struktur verirsə, CSS görünüşü idarə edir.

👨‍💻 Kim yaradıb?

CSS 1996-cı ildə Håkon Wium Lie tərəfindən yaradılmışdır.

🎯 Niyə yaradılıb?

HTML dizayn üçün kifayət etmirdi. CSS yaradıldı ki, saytın görünüşü ayrıca idarə olunsun.

🎨 Nümunə

body {
  background:black;
  color:white;
}

📐 Flexbox

.container {
  display:flex;
  justify-content:center;
}

✨ Animasiya

.box:hover {
  transform:scale(1.2);
}
