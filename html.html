📘 HTML TAM PRO KİTAB

📜 HTML nədir?

HTML (HyperText Markup Language) veb saytların strukturunu qurmaq üçün istifadə olunan əsas dildir. HTML olmadan heç bir sayt mövcud ola bilməz.

👨‍💻 Kim yaradıb?

HTML 1991-ci ildə Tim Berners-Lee tərəfindən yaradılmışdır.

🎯 Məqsədi

İnternetdə məlumat paylaşmaq və səhifələri strukturlaşdırmaq.

🧱 HTML əsas struktur

<!DOCTYPE html>
<html>
<head>
<title>Sayt</title>
</head>

<body>
<h1>Salam</h1>
</body>

</html>
