📄 HTML Tam Bələdçi

HTML nədir?

HTML (HyperText Markup Language) veb səhifələrin əsasını təşkil edən dildir. İnternetdə gördüyün bütün saytların strukturu HTML ilə qurulur.

👨‍💻 Kim yaradıb?

HTML 1991-ci ildə Tim Berners-Lee tərəfindən yaradılmışdır.

🎯 Niyə yaradılıb?

İnternetdə məlumatların paylaşılması və strukturlaşdırılması üçün yaradılmışdır. HTML olmadan sayt sadəcə mətn olardı.

🧱 HTML strukturu

<!DOCTYPE html>
<html>
<head>
<title>Başlıq</title>
</head>
<body>

<h1>Salam</h1>

</body>
</html>

🏷️ Əsas taglar

  • <h1> - Başlıq
  • <p> - Mətn
  • <a> - Link
  • <img> - Şəkil
  • <div> - Blok

💻 Real nümunə

<h1>Mənim saytım</h1>
<p>Bu mənim ilk səhifəmdir</p>
<a href="#">Kliklə</a>
